# 🌱 Plant Moisture Detector

A Python-based application that helps users monitor and manage plant moisture levels. The application allows users to record moisture values, store plant information, and visualize moisture trends through graphs.

## 🚀 Features

- User-friendly interface
- Add and manage plant information
- Store moisture readings in a SQLite database
- Display moisture history
- Generate moisture graphs
- Simple and lightweight application

## 🛠️ Technologies Used

- Python
- SQLite
- Matplotlib
- Tkinter / CustomTkinter
- Git & GitHub

## 📂 Project Structure

```
database.py    # Database operations
graph.py       # Graph visualization
main.py        # Main application
plant.py       # Plant management
theme.py       # UI theme
plants.db      # SQLite database
```

## ▶️ How to Run

1. Clone the repository
2. Install the required Python packages
3. Run:

```bash
python main.py
```
